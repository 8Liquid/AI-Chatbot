Example: AI Support Bot Embedded

This page demonstrates how the AI Support Bot plugin can be embedded in any website.

The chatbot widget appears in the bottom-right corner. Click it to start chatting!

How to Embed

To embed the chatbot in your website, simply include the CSS and JavaScript files:

<!-- In your HTML head -->
<link rel="stylesheet" href="styles.css">

<!-- Before closing body tag -->
<script src="renderer.js"></script>

The chatbot will automatically initialize when the page loads.
