# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - [YYYY-MM-DD]

### Added
- Initial release of AI Chatbot
- Highly customizable configuration system
- Support for custom colors, positions, and themes
- AI API integration support (OpenAI, Anthropic, custom)
- Local storage for message persistence
- Responsive design for all devices
- Keyboard shortcuts (Enter to send)
- Typing indicators
- Suggestion chips
- Custom avatars
- Callback hooks (onOpen, onClose, onMessage, onError)
- Dark theme support
- Zero dependencies - pure vanilla JavaScript
- Comprehensive documentation
- Integration examples for React, Vue, Next.js, WordPress, Shopify

### Features
- 40+ configuration options
- 4 position options (bottom-right, bottom-left, top-right, top-left)
- Customizable dimensions, colors, and styling
- Programmatic API (open, close, send, clear, updateConfig)
- Fallback knowledge base for offline responses
- Smooth animations and transitions
- Accessible design with ARIA labels
