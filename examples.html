1. Basic Setup

The simplest way to get started with default settings.

const chatbot = new AIChatbot();

2. Custom Colors

Customize the primary color and bubble colors.

const chatbot = new AIChatbot({ primaryColor: '#10b981', primaryHover: '#059669', userBubbleColor: '#10b981' });

3. Custom Position

Position the chatbot in different corners.

const chatbot = new AIChatbot({ position: 'bottom-left' });

4. Dark Theme

Use a dark color scheme.

const chatbot = new AIChatbot({ theme: 'dark', backgroundColor: '#1a1a1a', textColor: '#ffffff', botBubbleColor: '#2d2d2d' });

5. Custom Messages

Customize welcome message and suggestions.

6. Auto Open

Automatically open the chat when page loads.

const chatbot = new AIChatbot({ autoOpen: true, welcomeMessage: "Welcome! I'm here to help." });
